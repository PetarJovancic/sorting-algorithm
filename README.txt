How to for program execution:
1. Install python~=3.8 and set correct PATH of python.exe
2. Download "sorting" file and unzip file in a random directory
3. open execution.bat with notepadd++ or .txt editor 
4. change directory of execution.py file if necessary -> example: 
"C:\Users\Admin\AppData\Local\Programs\Python\Python38-32\python.exe C:\Users\Admin\Desktop\sorting\src\execution.py %*
pause"
5. Save and double click on execution.bat
6. "output.txt" will be generated with sorted values
